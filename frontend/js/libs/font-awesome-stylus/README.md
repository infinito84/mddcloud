Font Awesome Stylus
===================

Stylus Port for Font Awesome 4

* Updated to 4.2.0

## Installation
Import index.styl in your project and configure fonts path correctly ($fa-font-path)

## Source
http://fortawesome.github.io/Font-Awesome

## Bower support
`bower install font-awesome-stylus`
